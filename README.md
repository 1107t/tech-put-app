# プロジェクト概要

本システムは、講師側と受講生側に分かれており、学習ができたり、受講生同士のコミュニティ内で交流を深めるためのシステムです。

## 主な機能

### e-learning機能
1. 講師側: e-learning形式で投稿が可能で随時アップデートが可能
2. 受講生側: 閲覧のみ可能

### つぶやき機能（受講生と講師で出来るコミュニティ機能）
1. 講師（管理者）: 動画・テキスト・資料をアップロード
2. 受講生（ユーザー側）: 加入後、教材を閲覧・ダウンロードし、報告・質問・相談機能を通じて受講生/講師同士が交流可能
3. 管理者: ユーザー/コンテンツを監視し、不正利用や規約違反を管理画面から統制

### 記事投稿機能
- 講師側、受講生側でもマークダウン形式の記事が投稿可能

### YouTube投稿機能
- 受講生/講師が作成した動画をアウトプットできる仕様

### コミュニティ機能
- 受講生同士でお互いの近況や疑問などを投稿し合える

## 技術スタック

### フロントエンド
- 言語: TypeScript, JavaScript
- フレームワーク: React 18.x系（SPA）
- CSSフレームワーク: Tailwind CSS
- ビルドツール: Vite.js
- データベース: なし（APIのみ）
- テスト: なし（APIのみ）

### バックエンド
- 言語: Ruby 3.3.0
- フレームワーク: Ruby on Rails 7.0.8.1（APIモード）
- ビルドツール: Gemfile
- データベース: MySQL 8.0.35
- テスト: RSpec

### インフラ
- コンテナ: Docker でローカル・CI/CD 環境を統一

### インストール手順

```
npm run dev
```

## ルート設計（Routes）

### ユーザー用ルート
- /login: ログインページ
- /signup: ユーザー登録ページ
- /dashboard: ユーザーのダッシュボード

### 管理者用ルート
- /admin/login: 管理者ログインページ
- /admin/dashboard: 管理者ダッシュボード

## プロジェクトの目的

講師と受講生が効率的に学習・交流できるプラットフォームを提供し、教材の配信からコミュニティ形成まで一元管理できるシステムを構築すること。
